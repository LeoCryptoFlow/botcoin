# BotCoin (BTC) - 基于OpenClaw的加密货币系统

## 项目概述

BotCoin (BTC) 是一个实验性质的加密货币系统，旨在演示区块链技术和数字货币的基本原理。该项目仅供教育和研究目的。

## 核心特性

- 基于区块链的分布式账本
- 工作量证明(PoW)共识机制
- 数字签名验证交易
- 分布式网络节点
- 内置挖矿机制

## 技术栈

- Python 3.x
- Flask (轻量级API)
- SQLite (本地存储)
- SHA-256 (加密算法)
- RSA (数字签名)
